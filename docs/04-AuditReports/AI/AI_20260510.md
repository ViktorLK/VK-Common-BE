### 深度ロジック審査 (Deep Logic & State Evolution)

#### 実行パス推演: `VKChatSession.ChatAsync()`

```
1. ChatAsync(args, ct) 呼び出し
2. → _engine.SendAsync(_history, args, ct).ConfigureAwait(false)
3. → VKResult<VKChatMessage> 返却
4. result.IsSuccess → _history.Add(result.Value) → 履歴に蓄積
5. result.IsFailure → 履歴に追加されず、Result をそのまま返却
```

✅ 成功パス: 応答メッセージが履歴に正しく追加される
✅ 失敗パス: 失敗メッセージは履歴に混入しない。Result が呼び出し元に正しく伝播

#### 潜在的リスク

- ⚠️ `VKAiRequestContext.Usage` が `set` アクセサを持つ（`init` ではない）— immutability の一貫性に若干の違反
- ⚠️ `VKAIArgs.ModelName` プロパティが `ModelId` と重複 — 意図不明な二重定義

# Task: Refactor Configuration Section Naming to Namespace-Underscore Pattern
**ID**: CORE-015
**Status**: 🟡 Medium | #Debt
**Target**: `VK.Blocks.Core.Constants.VKBlocksConstants, IVKBlockMarker`
**Ref**: AP.04, BB.05

## 📝 Description
Implement a standardized transformation logic to convert logical building block identifiers (Namespace-style with dots, e.g., 'AI.SemanticKernel') into shell-safe configuration keys (Underscore-style, e.g., 'AI_SemanticKernel'). 

This task involves:
1. Adding a 'ToConfigKey' utility in 'VK.Blocks.Core' (e.g., in VKBlocksConstants).
2. Ensuring the 'BlockName' generated by Source Generators is sanitized or providing a 'ConfigSafeName' property.
3. Updating all BuildingBlock Option classes to use this transformation for their 'SectionName', ensuring maximum compatibility with environment variables in Linux/Docker environments.
